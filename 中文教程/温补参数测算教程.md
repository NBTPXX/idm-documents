# 温补参数测算教程

本教程用于针对性优化温补参数，降低温度漂移。

> **注意：** 本操作耗时较长（约 1 小时以上），如果当前不开启温补已能满足精度需求，则无需执行本操作。

## 一、配置数据采集宏

将以下宏粘贴到 `printer.cfg` 中：

```ini
[gcode_macro DATA_SAMPLE]
gcode:
  {% set bed_temp = params.BED_TEMP|default(90)|int %}
  {% set nozzle_temp = params.NOZZLE_TEMP|default(250)|int %}
  {% set min_temp = params.MIN_TEMP|default(40)|int %}
  {% set max_temp = params.MAX_TEMP|default(70)|int %}
  M106 S255
  TEMPERATURE_WAIT SENSOR='temperature_sensor IDM_coil' MAXIMUM={min_temp}
  M106 S0
  G28
  G0 Z1
  M104 S{nozzle_temp}
  M140 S{bed_temp}
  TEMPERATURE_WAIT SENSOR='temperature_sensor IDM_coil' MINIMUM={min_temp}
  IDM_STREAM FILENAME=/tmp/data1
  TEMPERATURE_WAIT SENSOR='temperature_sensor IDM_coil' MINIMUM={max_temp}
  IDM_STREAM FILENAME=/tmp/data1
  M104 S0
  M140 S0
  M106 S255
  G0 Z80
  TEMPERATURE_WAIT SENSOR='temperature_sensor IDM_coil' MAXIMUM={min_temp}
  M106 S0
  G28 Z0
  G0 Z2
  M104 S{nozzle_temp}
  M140 S{bed_temp}
  G4 P1000
  IDM_STREAM FILENAME=/tmp/data2
  TEMPERATURE_WAIT SENSOR='temperature_sensor IDM_coil' MINIMUM={max_temp}
  IDM_STREAM FILENAME=/tmp/data2
  M104 S0
  M140 S0
  M106 S255
  G0 Z80
  TEMPERATURE_WAIT SENSOR='temperature_sensor IDM_coil' MAXIMUM={min_temp}
  M106 S0
  G28 Z0
  G0 Z3
  M104 S{nozzle_temp}
  M140 S{bed_temp}
  G4 P1000
  IDM_STREAM FILENAME=/tmp/data3
  TEMPERATURE_WAIT SENSOR='temperature_sensor IDM_coil' MINIMUM={max_temp}
  IDM_STREAM FILENAME=/tmp/data3
  M104 S0
  M140 S0
```

## 二、采集数据

执行数据采集命令：

```gcode
DATA_SAMPLE BED_TEMP=<热床温度> NOZZLE_TEMP=<喷嘴温度> MIN_TEMP=<采集温度下限> MAX_TEMP=<采集温度上限>
```

各参数均为可选，不输入时将使用默认值：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `BED_TEMP` | 90 | 采集时热床温度 |
| `NOZZLE_TEMP` | 250 | 采集时喷嘴温度 |
| `MIN_TEMP` | 40 | 采集温度范围下限 |
| `MAX_TEMP` | 70 | 采集温度范围上限 |

**使用默认参数的最简形式：**

```gcode
DATA_SAMPLE
```

采集完成后会在 `/tmp/` 路径下生成三个数据文件：`data1`、`data2`、`data3`。

## 三、计算温补参数

1. 将生成的三个数据文件移动到用户目录下的 IDM 文件夹中
2. 执行计算脚本：

```bash
cd ~/IDM
~/klippy-env/bin/python arg_fit.py
```

> 执行前请确认当前使用的是最新版脚本包，如不确定请重新 `git clone`。

脚本计算量较大，需要等待一段时间。运行完成后会在 IDM 文件夹中生成：
- 三个温补参数
- 一张拟合效果图：`fit_result.png`

## 四、检查拟合效果

查看 `fit_result.png` 图片：

![fit_result](/imgs/fit_result.png)

- **第一行：** 原始数据
- **第二行：** 温补后的数据

拟合效果良好的标准：温补后的偏移量均控制在三位数以内。

## 五、写入配置

将脚本生成的三个参数复制到配置文件中的 `[idm]` 或 `[scanner]` 配置段下。

## 六、完成

参数写入后重启 Klipper，建议按主教程重新执行一次 `IDM_CALIBRATE` 校准步骤。
